Welcome to the HeyAPI Homepage!

This page is a HTML page.

This project, HeyAPI, is a simple web server built with Golang. It's a learning exercise to understand the fundamentals of creating web applications in Go, including serving static content like this page and handling API requests.

Feel free to explore the API endpoint! You can try accessing the /api route (e.g., using `curl http://localhost:8080/api`) to see a simple API response.
